# Images

Run `bash download-media.sh` from the project root to populate this folder with all 174 images from the WordPress media library.

## Key files needed immediately:
- Integrain-Site-Logo-2025.png       (navbar logo)
- Integrain-Foundation-Logo-2025.png (footer/og logo)
- Lotus-Logo-i-white.png             (white logo variant)
- home-hero-image.png                (home page hero)
- fire-ca.png                        (wildfire relief section)

## Source
All images were hosted at: https://integrain2026.test/wp-content/uploads/
Change the BASE variable in download-media.sh to your live domain before running.
